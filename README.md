#ServerMonitor

机房相关监控

## 功能简述
### 监测项目

+ 服务器详细信息

    + 流量
    + 开放端口
    + CPU MEM DISK
    + IP MAC
    + 远程历史
    + 进程状态
    + 温度
    + 查看系统日志（文件）

+ 网站状况
+ 机房温度
+ 是否断电
+ 监测历史

## 依赖包

+ pySerial
+ jinja2
+ pymongo
